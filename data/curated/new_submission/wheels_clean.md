| variable | class | description |
|:----------------------|:---------|:-----------------------------------------|
| name | character | Official or commonly used name of the ferris wheel. |
| height | double | Reported height of the ferris wheel in feet. |
| diameter | double | Reported diameter of the ferris wheel in feet. |
| opened | date | Date when the ferris wheel was first opened to the public. |
| country | character | Country where the ferris wheel is located, standardized to full country name. |
| location | character | City or specific site where the ferris wheel is located. |
| number_of_cabins | double | Total number of passenger cabins on the ferris wheel. |
| passengers_per_cabin | double | Number of passengers that can be seated in each cabin. |
| seating_capacity | double | Total seating capacity of the ferris wheel based on cabins and passengers. |
| hourly_capacity | integer | Estimated number of passengers the ferris wheel can carry per hour. |
| ride_duration_minutes | double | Average duration of a single ride in minutes. |
| climate_controlled | character | Indicates if cabins are climate controlled |
| status | character | Current operational status of the ferris wheel |
| construction_cost_million_usd | character | Reported construction cost in millions of USD |
